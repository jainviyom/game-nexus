# NEXUS — Core Recalibration

A single-file, no-backend browser puzzle game. Solve number, logic, and pattern puzzles to recharge a dormant reactor core — chain correct answers for combo multipliers, push stability to 100%, and climb the operator ranks.

**Play it live:** https://jainviyom.github.io/game-nexus/

## Run locally

```bash
open index.html
```

No build step, no dependencies, no server required.
